## Healthcare Data Analytics

## Project Overview
This project analyzes healthcare data using Excel, SQL, Power BI, and Tableau to identify patient trends, treatment costs, doctor performance, and hospital operations insights.

## Tools Used
- Excel
- SQL
- Power BI
- Tableau

## KPIs
- Total Patients
- Total Doctors
- Total Visits
- Average Age of Patients
- Top 5 Diagnosed Conditions
- Follow-Up Rate
- Treatment Cost Per Visit (Average)
- Total Lab Tests Conducted
- Percentage of Abnormal Lab Results
- Doctor Workload (Average Patients Per Doctor)
- Total Revenue = Sum(Treatment Cost) + Sum(Visit Charges)

## Insights
- Identified high-performing departments and doctors
- Analyzed patient visit and diagnosis trends
- Evaluated follow-up and treatment effectiveness
- Improved healthcare resource planning insights
- Monitored revenue and treatment costs efficiently
