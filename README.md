# Healthcare Data Analytics

## Project Overview
This project analyzes healthcare data using Excel, SQL, Power BI, and Tableau to identify patient trends, treatment costs, and department performance.

## Tools Used
- Excel
- SQL
- Power BI
- Tableau

## KPIs
- Total Patients
- Treatment Cost
- Department Performance
- Revenue Analysis

## Insights
- Identified high-performing departments
- Analyzed patient trends
- Improved resource planning insights

## Files Included
- Excel Dataset
- SQL Queries
- Power BI Dashboard
- Tableau Dashboard
- Project Presentation
